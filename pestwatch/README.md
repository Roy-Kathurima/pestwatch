PestWatch - community pest reporting

How to run:
- install requirements.txt
- set ADMIN_SECRET env var (optional, default 'letmein')
- push to Render or run locally: python app.py

Features:
- Register / Login
- Submit reports with image + device location
- Admin unlock via secret on landing page
- Admin map + approve reports + export CSV
- Login logs stored
- Password reset via security question
